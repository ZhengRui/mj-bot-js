This project is js version, and to create variation/scale, just reply "v1"/"s1".

You need to create a `config.json` file:

```json
{
  "token": "*",
  "clientId": "*",
  "guildId": "*",
  "userToken": "*",
  "MJID": "*"
}

```

Connect this project to runway.app to host the bot.
